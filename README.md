# date_range_process
日付の範囲を加減算する処理

# dates_add
- 日付範囲群に、日付範囲群を加えた範囲群を返す。
- （まだ作っていない）

# dates_subtract
- 日付範囲群から、日付範囲群を除外した範囲群を返す。

## 例
- 元の日付範囲
  - 2023/1/1～2023/3/31
  - 2023/5/10～2023/7/10
- 除外する範囲
  - 2023/3/15～2023/6/10
- 結果
  - 2023/1/1～2023/3/14
  - 2023/6/11～2023/7/10
